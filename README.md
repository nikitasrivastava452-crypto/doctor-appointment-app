# doctor appointment app

Tech Stack
- JavaScript
- Axios
- JSON Server (mock backend)

## Run the project

1. Install dependencies
npm install

2. Start backend
npx json-server --watch backend-app/db.json --port 3000

3. Start frontend
npm run dev
